# Apexcify-Internship-Task-4-Basic-Chatbot-Creation
# Task Overview
This project is a simple rule-based chatbot built using Python. It interacts with users through predefined responses based on user input.

# Features
Continuous conversation using loops
Responds to greetings and common questions
Handles exit commands gracefully
Simple and interactive text-based interface
# How It Works
The chatbot starts and waits for user input
User messages are converted to lowercase for consistency
The program checks for keywords using conditional statements
Responds accordingly
Ends conversation when user types "bye"

# Sample Output
Chatbot ready! Say 'bye' to exit.

- Chatbot: Hi!
- Chatbot: I'm doing great, thanks! How about you?
- Chatbot: My name is GeorgyBot, how may I help you?
- Chatbot: Goodbye! Have a nice day!

# Tools & Technologies
- Jupyter notebook
- Conditional Statements (if-elif)
- Loops
- Input/Output Handling
